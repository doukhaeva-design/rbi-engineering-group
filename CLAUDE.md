# Правила для проекта RBI Engineering Group

## Архитектура — НЕЛЬЗЯ менять без разрешения
- Структуру папок: `app/`, `components/`, `public/`
- Маршруты страниц: `/about`, `/services`, `/modular`, `/projects`, `/licenses`, `/contacts`, `/policy`
- Компоненты: не добавлять, не удалять, не переименовывать без разрешения
- `LanguageContext.tsx` — логику мультиязычности не трогать
- `layout.tsx` — структуру не менять
- `package.json` — зависимости не добавлять и не удалять
- API: `app/api/contact` — не трогать

## Дизайн — МОЖНО менять по просьбе
- CSS-модули каждого компонента (`*.module.css`)
- `globals.css` — только CSS-переменные и стили, не структуру
- Цвета, отступы, шрифты, размеры, анимации
- Текст и контент внутри компонентов
- Изображения в `public/`

## Дизайн-система (не нарушать без причины)
Цвета определены в `globals.css` через переменные:
- Фон: `--background: #ffffff`
- Текст: `--text-main: #111827`, `--text-secondary: #374151`
- Акцент синий: `--primary-color: #2F5D9F`
- Акцент золото: `--premium-copper` (gradient)
- Серый: `--steel-color: #64748b`

Шрифты:
- Семейство: Inter (Google Fonts, кириллица + латиница)
- H1: `--font-h1: 72px` (мобайл: 36px)
- H2: `--font-h2: 32px` (мобайл: 28px)
- H3: `--font-h3: 20px`
- Текст: `--font-text: 18px` (мобайл: 16px)

Сетка:
- Макс. ширина: `--max-width: 1280px`
- Отступы по бокам: `--padding-x: 32px` (мобайл: 24px)
- Вертикальные отступы: `--vertical-gap: 100px` (мобайл: 60px)

## Компоненты (секции главной страницы)
- `Hero` — первый экран с заголовком и фото
- `AboutSection` — профиль компании со счётчиками
- `Activities` — услуги компании
- `ModularPreview` — секция модульных сооружений
- `WhereUsed` — сферы применения
- `KeyCompetencies` — почему выбирают нас
- `KeyProjects` — ключевые проекты
- `Partners` — партнёры
- `CTA` — призыв к действию

UI-элементы:
- `Button` — стандартная кнопка
- `RevealOnScroll` — анимация появления при прокрутке
- `LanguageSwitcher` — переключатель языка
- `BackButton` — кнопка назад

## Правила при изменениях
- Всегда использовать CSS-переменные из `globals.css` — не хардкодить цвета
- Соблюдать отступы из переменных `--padding-x` и `--vertical-gap`
- Мобильная адаптация обязательна при любом изменении дизайна
- Не дублировать стили — сначала проверить есть ли нужная переменная
